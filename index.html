# LMPR-PS

This is the implementation of LMPR-PS using PyTorch.

## Requirements

* CUDA 10.2
* PyTorch 1.6
* Other requirements
`pip install -r requirements.txt`

## Test

1. Download dataset, and extract.
2. Convert `mat` format labels to `json` format using MATLAB script `scripts/mat2json.m`.
3. Test

```(shell)
python test.py --label_directory $LABEL_DIRECTORY --image_directory $IMAGE_DIRECTORY --detector_weights $DETECTOR_WEIGHTS
```

`LABEL_DIRECTORY` is the directory containing testing json labels.
`IMAGE_DIRECTORY` is the directory containing testing jpg images.
`DETECTOR_WEIGHTS` is the trained weights of detector.
View `config.py` for more argument details.
